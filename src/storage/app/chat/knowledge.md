## Projects

### Portfolio Platform (this website)
A production-grade microservices system built with Laravel 12, deployed on Docker with Kubernetes manifests. The platform consists of 6 independent services:
- **Frontend** — main web app with blog, user panel, contact form
- **Blog API** — REST API for posts, comments, categories, tags, newsletter
- **SSO** — OAuth2 authorization server (Laravel Passport)
- **Admin** — admin panel (FilamentPHP)
- **Users** — user management and RBAC
- **Analytics** — page view and like tracking

Infrastructure includes Traefik, RabbitMQ, Prometheus, Grafana, Loki, and Promtail.

The blog supports multilingual content (Polish/English), markdown editing, code syntax highlighting, newsletter subscriptions, and a comment moderation workflow.
